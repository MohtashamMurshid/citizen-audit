CitizenAudit Agent

Crowdsourced + AI-powered field verification of ISI products

How it works:

Citizen uploads photo of a product from a local shop
OCR Agent extracts ISI number
Verification Agent checks BIS database
Geolocation Agent maps fake product hotspots across India
Builds a live heatmap of counterfeit product concentration
Escalation Agent batches reports and sends to BIS regional offices

Why it's novel: Crowdsourced intelligence + agentic automation + real policy impact
